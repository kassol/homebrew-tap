# kassol/homebrew-tap

Homebrew tap for kassol CLI tools.

## Available Formulae

- `brewcli` - Interactive Homebrew TUI built with Bun, TypeScript, and Ink
- `skills-tui` - TUI for managing global AI agent skills

## Usage

```bash
brew tap kassol/tap
brew install brewcli
brew install skills-tui
```

## Source Repositories

- `brewcli`: https://github.com/kassol/brewcli
- `skills-tui`: https://github.com/kassol/skills-tui
